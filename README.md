# Elgml — بورتفوليو

موقع بورتفوليو بسيط لعرض أعمال `Elgml` كمعلق وصانع إعلانات ومخرج أفلام بالذكاء الاصطناعي.


تواصل: mohamedelgamal723@gmail.com

إضافة الفيديوهات:

1. ضع ملفات الفيديو `project1.mp4`, `project2.mp4`, `project3.mp4` داخل مجلد `videos` في جذر الموقع.
2. أضف صور غلاف بنفس الأسماء (`project1.jpg`, ...). يمكنك استخدام أي أسماء ولكن حدث المسارات في `index.html` إذا غيّرتها.
3. راجع `/elgml-site/videos/README.md` لنصائح الترميز وحجم الملفات.

نشر على GitHub Pages (خطوات سريعة):

1. أنشئ مستودع جديد على GitHub.
2. انسخ محتويات مجلد `elgml-site` إلى جذر المستودع أو ضعها داخل مجلد `docs` إذا تفضّل ذلك.
3. ادفع (push) التغييرات إلى الفرع `main`.
4. من إعدادات المستودع اذهب إلى `Pages` واختر مصدر النشر: `main` أو `gh-pages` أو `docs`، ثم انتظر نشر الموقع.

بدائل الاستضافة المجانية: Netlify أو Vercel — اجعل المستودع متصلاً وسيتم النشر تلقائياً مع كل دفع.

تفاصيل تسليم الموقع إليك:

- هيكل المشروع جاهز داخل مجلد `elgml-site`.
- أضفت معرض فيديو يعمل مكانياً (ملفات MP4 داخل `videos`).
- ارفع ملفات الفيديو وصور الغلاف ثم ادفع المستودع، وسأقدّم لك رابط الموقع إذا رغبت.
